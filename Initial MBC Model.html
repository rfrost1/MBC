Model

For reference, I chose the bmi value at index and change instead of weight, but weight can be used as well to see the effect. The variables beginning with ‘main’ are the confounders from previous papers I believe, although I am not 100% sure whether they were used or not. I used a categorical variable to describe family history, but as discussed we could go a number of ways with the information we have.

mbc_model <- glm(case ~ indexage + clean_acorn_cat + maingynae + mainradiotherapy + maintesticular + mainsexhormone + selfinfertile + index_bmi_est + index_bmi_est_missing + bmichangeper + bmichangeper_missing + familybc_spec, data = rf, family = "binomial")
